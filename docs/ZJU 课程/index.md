# ZJU 课程

这里整理浙江大学软件工程相关课程的学习记录、课程体验和复习资料。内容以个人经历为主，适合选课前了解课程风格，也适合期末复习时快速定位资料。

## 专业必修课

- [CSP](./专业必修课/CSP.md)：程序设计能力训练与认证相关记录。
- [DB](./专业必修课/DB.md)：数据库系统课程笔记。
- [DM](./专业必修课/DM.md)：离散数学相关内容。
- [FSE](./专业必修课/FSE.md)：软件工程基础课程记录。
- [OOP](./专业必修课/OOP.md)：面向对象程序设计课程记录。
- [ADS](./专业必修课/ads.md)：高级数据结构相关内容。
- [FDS](./专业必修课/fds.md)：基础数据结构相关内容。

## 专业选修课

- [CG](./专业选修课/CG.md)：计算机图形学。
- [CTF101](./专业选修课/CTF101.md)：CTF 入门课程记录。
- [NA](./专业选修课/NA.md)：数值分析。
- [信安原](./专业选修课/信安原.md)：信息安全原理。
- [计算医疗](./专业选修课/计算医疗.md)：计算医疗相关课程记录。
- [软测](./专业选修课/软测.md)：软件测试。
- [软需](./专业选修课/软需.md)：软件需求工程。
